# ▶️ How To Run

## 1 Clone the repository

```bash
git clone https://github.com/flavio083/pygame-fighting-game.git
```

## 2 Enter the folder

```bash
cd pygame-fighting-game
```

## 3 Install dependencies

```bash
pip install pygame
```

## 4 Run the project

```bash
python main/main.py
```
